Act as a systems integration engineer.

Your task is to verify that ALL components of this application work together correctly.

Steps:
1. Identify all integration points:
   - frontend ↔ backend
   - backend ↔ database
   - backend ↔ external APIs / AI models

2. For each integration:
   - validate request/response contracts
   - detect mismatches in schema
   - check error handling

3. Simulate full workflows:
   - user signup → login → feature usage → result display
   - API failure scenarios
   - partial system outages

4. Identify:
   - broken flows
   - silent failures
   - incorrect assumptions between components

5. Output:
   - failing workflows
   - exact integration gaps
   - fixes with code-level suggestions
